How to play

  • Work as a team. Choose a level (A / B / C).
  • Group the 16 words into 4 groups of 4.
  • Show your grouping to the teacher for points.
Goal: Make 4 groups of 4. No need to name the categories.

Teacher check: 4 correct groups = 1 point.

Goal: Make 4 groups of 4 and write a short category name (1–3 words) for each group.

Teacher check: correct groups + reasonable labels = 2 points.

Goal: Make 4 groups of 4, name each category, then write one clear example sentence using one word from any group.

Teacher check: correct groups + clear example sentence = 3 points.

Teacher scoring: Level A = 1 point, Level B = 2 points, Level C = 3 points. Optional: +1 collaboration token (cap per team or class).
